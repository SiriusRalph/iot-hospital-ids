# Code source

Ce dossier contient les modules Python réutilisables.

## Fichiers
- `preprocessing.py` : Fonctions de chargement et prétraitement des données
- `models.py` : Définition et entraînement des modèles
- `utils.py` : Fonctions utilitaires (visualisation, métriques, etc.)
- `__init__.py` : Fichier pour faire de "src" un package Python
